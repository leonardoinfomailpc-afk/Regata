# Regata

## 🚀 Avvio rapido

1) Installa le dipendenze:

```bash
npm install
```

2) Avvia il server di sviluppo (Vite):

```bash
npm run dev
```

3) Apri il browser sull’URL che Vite stampa in console (di solito `http://localhost:5173`).

---

## 🧠 Come funziona (sintetico)

- L’app è una **SPA React** costruita con **Vite**.
- Le pagine/competenze sono sviluppate in **TypeScript/TSX** sotto la cartella principale.
- Vite avvia un server di sviluppo con reload automatico mentre modifichi il codice.
- La build di produzione viene generata con `npm run build`.
